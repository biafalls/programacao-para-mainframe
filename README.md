# Programação para Mainframe

## Sobre

Este repositório reúne exercícios desenvolvidos durante a disciplina de **Programação para Mainframe**, utilizando a linguagem **COBOL**.

Os exercícios foram organizados e publicados com o objetivo de documentar meu aprendizado e registrar minha evolução no desenvolvimento de aplicações para ambiente mainframe.

## Conteúdos abordados

- Sintaxe da linguagem COBOL
- Manipulação de variáveis
- Estruturas condicionais
- Estruturas de repetição
- Processamento de arquivos
- Resolução de problemas utilizando COBOL

## Tecnologias utilizadas

- COBOL
